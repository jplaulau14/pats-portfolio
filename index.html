All-in-One Short Stay Rental Management Platform

A comprehensive platform that empowers property managers to effectively manage their short stay rental business by providing data insights, automated pricing and minimum stay updates, and intelligent recommendations.

Key Features:

  • Interactive dashboard displaying key metrics such as revenue, profit, and Average Daily Rate (ADR) for data-driven decision making.
  • Seamless integration with popular short stay rental platforms like Airbnb and Booking.com, enabling centralized management and synchronization of pricing and minimum stay settings.
  • Dynamic pricing and minimum stay recommendation engine that analyzes historical data, market trends, and seasonal patterns to provide intelligent insights for optimizing revenue.
  • Intuitive user interface for effortless tracking, monitoring, and adjustment of pricing and minimum stay settings across multiple properties and platforms.

Technologies Used:

  • Full Stack: Python Flask, AWS Lambda, Serverless Framework, React, PostgreSQL, GitHub Actions, Docker
  • Data Engineering: Apache Airflow, Amazon Redshift, Amazon Athena, AWS Lake Formation, Supabase, PySpark, Amazon S3, Terraform
  • Machine Learning: Python, PyTorch, Tensorflow, Amazon SageMaker

Developed a scalable and modular architecture that seamlessly integrates full-stack development, data engineering, and machine learning components. Built a responsive frontend using NextJS and React, powered by a robust backend API using Flask and AWS Lambda. Implemented a data processing pipeline using Apache Airflow for data ingestion, transformation, and storage in Amazon Redshift and Amazon S3. Utilized PyTorch and Tensorflow to build intelligent recommendation models for dynamic pricing and minimum stay optimization. Deployed the machine learning models using Amazon SageMaker for efficient inference. Ensured code quality and automated deployments using GitHub Actions and containerization with Docker.
